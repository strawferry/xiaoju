# 🚀 快速开始指南

## 🎯 问题解决

如果你看到了 **"数据加载失败，请确保data.json文件存在且格式正确"** 的错误，请按以下步骤操作：

### ✅ 最简单的解决方案

**直接打开这个文件：**
```
charging_dashboard_embedded.html
```

**操作步骤：**
1. 在文件管理器中找到 `charging_dashboard_embedded.html`
2. 双击打开
3. 浏览器会自动显示数据面板

### 🔧 备选方案

如果上面的方法不行，试试这些：

#### 方案1：使用本地服务器
```bash
# 打开终端/命令行
cd /Users/m4/Desktop/xiaoju
python3 -m http.server 8000

# 然后在浏览器访问：
# http://localhost:8000/charging_dashboard.html
```

#### 方案2：VS Code用户
1. 安装 "Live Server" 插件
2. 右键 `charging_dashboard.html` 
3. 选择 "Open with Live Server"

#### 方案3：其他工具
- **Chrome浏览器**：安装 "Web Server for Chrome" 扩展
- **Node.js**：`npx http-server`
- **PHP**：`php -S localhost:8080`

## 🎨 网页功能

打开后你将看到：

### 📊 总览页面
- 37条充电记录的统计
- 本月vs上月对比
- 常用充电站识别
- 月度消费趋势图
- 充电站分布图

### 📋 列表页面
- 完整订单表格
- 搜索功能（订单号、充电站）
- 筛选功能（充电站、月份）
- 数据导出CSV

### 📅 时间线页面
- 按月份分组的时间轴
- 可视化充电记录
- 点击查看详情

### 📈 分析页面
- 极值分析（最高/最低消费等）
- 充电时段分布
- 费用构成分析
- 充电效率排名
- 月度频率统计
