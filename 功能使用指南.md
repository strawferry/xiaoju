# 🔋 电车充电数据面板 - 功能使用指南

## 📱 界面概览

打开 `charging_dashboard_modified.html` 后,您将看到:

### 1️⃣ 顶部统计卡片
- **总订单数**：所有充电订单总数
- **最终付款总额**：实际支付的总金额（使用 paymentFee）
- **总电量**：累计充电量（kWh）
- **平均单价**：平均每度电的价格

### 2️⃣ 标签页导航
- 📋 **列表视图**：卡片式展示所有充电记录
- 📅 **日历视图**：按日历形式展示充电记录
- 📊 **数据分析**：月度统计、站点分布、总体概览

### 3️⃣ 筛选功能
- **搜索框**：搜索站点名称或订单号
- **月份筛选**：选择特定月份
- **站点筛选**：选择特定充电站

## 🎯 核心功能亮点

### 1️⃣ 费用透明化
- **最终付款**：实际支付金额（paymentFee）
- **原始费用**：totalFee（可能包含优惠）
- **优惠金额**：自动计算差额
- **费用分解**：电费 + 服务费

### 2️⃣ 三层钻取分析
```
月度统计 → 点击月份
    ↓
月度日历 → 点击日期
    ↓
当日订单 → 点击订单
    ↓
完整详情
```

### 3️⃣ 智能汇总
- 按月自动统计
- 按日自动分组
- 自动计算优惠
- 实时筛选响应

## 💡 使用技巧

### 快速查看月度消费
1. 切换到"📊 数据分析"标签
2. 直接查看月度统计卡片
3. 点击"查看详情"进入日历

### 分析某天的充电情况
1. 进入月度日历视图
2. 找到目标日期（有红色标记）
3. 点击日期查看当日所有订单

### 对比不同月份
1. 在月度统计页面查看各月数据
2. 点击不同月份卡片查看详细日历
3. 注意观察"优惠金额"的变化

### 查看费用构成
1. 在列表视图查看每条记录的费用分解
2. 在日历弹窗中查看当日费用汇总
3. 在月度总结中查看月度费用构成

## 🔍 数据说明

### 关键字段
| 字段 | 说明 | 用途 |
|------|------|------|
| `paymentFee` | 最终付款金额 | 统计实际收入 |
| `totalFee` | 原始总费用 | 对比优惠 |
| `eleFee` | 电费 | 费用分解 |
| `servFee` | 服务费 | 费用分解 |
| `totalPower` | 充电量 | 计算单价 |
| `chargeTime` | 充电时长(秒) | 转换为分钟 |

### 价格计算
- **当日单价** = 当日付款 ÷ 当日电量
- **月度单价** = 月度付款 ÷ 月度电量
- **优惠金额** = 原始费用 - 最终付款
